웹 서버에서 웹 텍스트 데이터 읽는 예제, [블로그](https://agvim.com/2019/02/09/web-server-read-web-text-data/)에서 간단한 설명을 볼 수 있다.

## Install

    git clone https://github.com/afrontend/readCard.git
    cd readCount
    npm install

## Run
    npm start
